## 5G CAT-RAT

5G/4Gのキャリアアグリゲーション（CA）状態や、基地局の物理パラメータをリアルタイムで可視化・ログ保存する解析ツールです。

## スクリーンショット
* 通知アイコン：実際の接続状況をリアルタイムに反映する通知アイコンRAT表示
* メイン画面（テキストタブ）：RSRP, RSRQ, SINR, CQI, TAなどの詳細一覧 。
* グラフ画面（積層表示）：各セルの帯域幅（MHz）と品質（QL）を視覚化した画面 。

## 動作要件
* OS：Android 11以上  
* 必須アプリ
  * Shizukuアプリが実行されていること。（dumpsys 取得に使用）  
* 必要な権限
  * 位置情報：基地局情報の詳細取得、およびCSVログへの座標記録に使用します。  
  * 電話 (端末の状態)：接続中のSIM情報や基地局情報の取得に使用します。  
  * 通知：通知ドロワーでのリアルタイム表示に使用します。  
  * 他のアプリの上に重ねて表示: オーバーレイ表示機能を使用する場合に必要です。  

## 主な機能
* 5G NSA/SA および 4G CA状態のリアルタイム判別。  
* Android標準APIでは取得困難な物理レイヤー情報の dumpsys 解析による補完。  
* 品質スコア（QL）の独自算出と理論上限速度の推定。  
* キャリアや周波数に基づくバンド名の動的推測。  
* Googleマップ等での解析に使えるかもしれない位置情報付きCSVログ保存 。

## 機能詳細
### 1.通知アイコン  
以下のパターンで通知アイコンを表示します。  
* 標準モード：5G+ / 5G / 4G+ / 4G / 緊急  
* 詳細モード：5GSA / 5転SA / 5GNSA / 5転NSA / 4G+eLTE / 4GeLTE / 4G+ / 4G / 緊急

### 2.通知ドロワー
* 1行目：接続パターン、全セルの合計帯域幅と独自計算の品質帯域幅、モデムのLink Capacity  
* 2行目：PCellの詳細データ バンド名、帯域幅、RSRP、RSRQ、SINR、CQI、TA  
* 3行目以降：SCellの詳細データ  
※表示項目は詳細設定である程度選択調整できます。  

### 3.オーバーレイ表示
* 1行目：接続パターン、全セルの合計帯域幅と独自計算の品質帯域幅、モデムのLink Capacity  
* 2行目：Ping値、接続している各セルの簡易情報(バンド名、品質値、電波混雑度)  
※表示項目は詳細設定である程度選択調整できます。  

### 4.詳細カード画面
取得できたデータの詳細情報を表示します。  
データが取得できなかった場合はハイフンになります。  
Pixel 10 Pro XLでは割と取得できましたが、  
arrows 5GではCQI等、取得できないデータがいくつかありました。  
端末やOSによって、取得できるデータできないデータがあります。  
表示されなかったらたぶんダンプデータに載っていないので、取得が難しいです。  
ダンプデータ確認用に、「ﾀﾞﾝﾌﾟ」ボタンを用意しています。  
Download/5GCATRAT のフォルダに保存されます。  
表示項目は詳細設定である程度選択調整できます。
近隣セルもオプションで表示可能です。(デフォルトはオフ)  

### 5.グラフ画面
枠が物理的な帯域幅、塗ってある部分が品質値を乗算した**品質帯域幅**です。  
理論値と実態のギャップが一目で分かりやすいように積層型のグラフにしました。  
品質帯域幅は、以下のような計算例で表示しています。

高さの初期値は100MHz幅で、物理的な合計帯域幅が100MHzを超えると、自動で高さを調整してくれます。

### 6.統計画面
アプリが稼働中(監視オン)であり、かつ画面オフになっていない時間で計測しています。  
PCell、SCellどちらで接続していたかも表示しています。  

### 7.タイムライン
PCellのハンドオーバーが発生した時刻を記録しています。  
パケ詰まりや、SA接続をしたときにも記録が入るようになっています。  

### 8.詳細設定
・一応デュアルSIMに対応しています。  
・他、表示非表示は割と選べるようにしているので、自由に好みで変えてください。  
・TA値が取得できない場合は、RSRPから推定距離を算出するようになっています。  
　送信電力と伝搬環境は、状況に合わせて設定してみてください。  
・マップログ用のCSVを出力できるようになっています。(出力項目がこれでいいのかわかりませんが)  
・Ping監視で、パケット遅延、パケ詰まりを判定できる機能があります。Ping飛ばす先は自由入力欄を追加しました。  
　小窓に表示をオンにすると、オーバーレイ表示の2行目左端にPing値が表示されます。  
　基本的にはモバイルネットワーク接続時用の機能ですが、Wi-Fi接続時にもPingを飛ばすオプションを追加しました。  
　公共Wi-Fiしかで役立つ...かもしれません。
